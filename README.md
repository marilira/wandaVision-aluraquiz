## Setup

- Clone este repositório
```bash
git clone https://github.com/marilira/wandaVision-aluraquiz.git
```
- Instale as dependências
```bash
yarn install
#ou
npm install
```
- Rode o projeto
```bash
yarn dev
#ou
npm run dev
```
